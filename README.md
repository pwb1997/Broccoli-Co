## Getting Started

### Prerequisite

-   Install [Node.js v16](https://nodejs.org/en/)
-   Install [Yarn](https://yarnpkg.com/) globally

### Install Dependencies

```
yarn install
```

### Build

```
yarn build
```

### Dev

```
yarn dev
```

### Test

Jest

```
yarn test
```

Jest watch

```
yarn test:watch
```

Jest coverage

```
yarn test:coverage
```
